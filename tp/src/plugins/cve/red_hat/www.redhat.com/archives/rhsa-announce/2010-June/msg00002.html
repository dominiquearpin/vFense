1. Summary:

This is the notification of the End of Life plans for Red Hat Directory
Server 7.1.

2. Description:

On June 7, 2010, per the life-cycle support policy, version 7.1 of Red Hat
Directory Server will exit Production Phase 2, marking the end of its
support by Red Hat. Please reference the support policy here: 
https://www.redhat.com/security/updates/directory/

For migration planning, please note the following dates:

* June 7, 2010 - Red Hat Directory Server 7.1 will end normal product
support. Only Severity 1 and security issues and migration assistance will
be addressed. Directory Server instances will continue to operate, but all
customers are encouraged to migrate in a timely manner.

* September 7, 2010 - Red Hat Directory Server 7.1 active status will be
terminated. Directory Server instances will continue to operate, but
technical support issues will no longer be addressed, and security errata
and bug fixes will no longer be released.

3. Solution:

How to proceed:

* All Red Hat Directory Server 7.1 users must plan to migrate to a newer
version prior to September 7, 2010.

* If you have a Technical Account Manager, contact that person immediately
to discuss migration plans.

* Otherwise, contact support for assistance: 
https://www.redhat.com/apps/support/

* Alternatively, Red Hat Consulting can be engaged to assist with a smooth
migration: 
http://www.redhat.com/consulting/

It is critical to ensure you have a plan to migrate off Red Hat Directory
Server 7.1 prior to September 7, 2010.

4. References:

http://www.redhat.com/security/updates/classification/#low
